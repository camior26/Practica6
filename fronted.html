Definición de Frontend

El frontend es la parte de una aplicación o sitio web con la que los usuarios interactúan directamente. Se encarga de la presentación visual, la estructura y la experiencia de usuario, asegurando que la información y las funcionalidades sean accesibles de manera intuitiva.

Lenguajes Esenciales

  • HTML (HyperText Markup Language): Define la estructura y el contenido de una página web.
  • CSS (Cascading Style Sheets): Se utiliza para dar estilo, colores, tipografía y diseño responsivo a la página.
  • JavaScript (JS): Permite la interactividad, como animaciones, validaciones de formularios y actualizaciones de contenido sin recargar la página.

Frameworks y Librerías

  • CSS Frameworks: Bootstrap, Tailwind CSS.
  • JavaScript Frameworks/Librerías: React, Vue.js, Angular.

Principios de diseño y experiencia de usuario (UX/UI)

  • Responsive Design: Adaptar la interfaz a distintos dispositivos y tamaños de pantalla.
  • Accesibilidad (a11y): Asegurar que personas con discapacidades puedan usar el sitio.
  • Usabilidad: Interfaces intuitivas y fáciles de navegar.

Herramientas y Entorno de Desarrollo

  • Versionado: Git y plataformas como GitHub.
  • Preprocesadores y herramientas de automatización: Sass, Webpack, Vite.
  • APIs y AJAX: Comunicación con el backend sin recargar la página.
